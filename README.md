# -TFUG-Mysuru-Water-Quality-Prediction
In this competition, our aim is to train a machine learning model based on the water quality data provided in the training file and further predict the quality estimation result for the test dataset. This competition is hosted by ML GDEs or TFUGs, sponsored by Google Developers.
<br><center><img src="images/water.png" width=100%></center>

#### Methods :point_down:
The implemented solution can be summarized in the following core steps:
* LGBMRegressor
* KFold as a bagging approach
* Ensemble learning
